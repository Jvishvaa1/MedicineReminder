# Smart Medicine Reminder Android App

📱 Android app to remind users to take medicines on time.

## Features

* Add medicines with time and date
* Select specific days (Mon–Sun / Everyday)
* Notifications with "Taken" button
* Dashboard (Total, Taken, Missed)
* Simple login screen

## Tech Used

* Kotlin
* Android Studio
* SQLite
* AlarmManager
